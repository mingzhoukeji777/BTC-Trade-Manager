# BTC交易管理系统

当前版本：`v0.6.8 选择/平仓类型/主仓编号修复版`

## v0.6.8 修复/新增

### 1. 图1说明

图1是 Hermes 的文件修改校验提示，不是软件错误。

意思是：之前某次 `patch` 尝试失败过，因为匹配内容出现了多处，Hermes 提醒要用 `git status` 或 `read_file` 确认文件是否真的修改成功。

后续修改已经通过脚本方式完成，并经过编译、运行和 GitHub Release 验证。

### 2. 表格默认未选中

所有表格加载后默认清空选中状态。

### 3. 双击表格空白处取消选中

鼠标选中表格行后，双击表格外/空白区域，会清空选中状态。

### 4. 新建主仓编号减少到 10 条

主仓编号下拉从 M1-M50 改为：

```text
M1 - M10
```

### 5. 修复平仓类型未写入

平仓时选择：

```text
止损
成本止损
移动止盈
手动平仓
```

会写入数据库，并在已平仓模块的 `平仓类型` 列显示。

### 6. 已平仓右键修改支持平仓类型

在已平仓模块右键修改，可以修改：

```text
平仓类型
平仓时间
平仓订单号
备注
```
