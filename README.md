# BTC交易管理系统

当前版本：`v0.6.6 主控/选中/平仓类型/主仓树修复版`

## v0.6.6 修复/新增

### 1. 修复主控界面灰色大空白

上一版总控页因为列宽模式改成固定，导致表格只占左侧一小块，右边大面积灰色。

现在总控页恢复为填满式表格，并把背景改成白色。

### 2. 主仓平仓后从主仓M隐藏

主仓右键平仓后：

```text
主仓M 页面不再显示该主仓
主仓树不再显示该主仓
已平仓 页面显示该主仓
```

### 3. 表格和主仓树选中效果优化

表格选中改为淡黄色，不再使用刺眼蓝底，也不会遮挡红绿盈亏字体。

主仓树选中也改为淡黄色高亮边框，更容易看出当前选中项。

### 4. 持有中状态使用红色球

主仓树状态球：

```text
持有中：红色
成本止损：黄色
移动止盈：绿色
其他：灰色
```

旧数据中的 `已开仓 / 持仓中` 会在启动时自动迁移为：

```text
持有中
```

新增主仓 / 加仓 / 对冲默认状态也改为：

```text
持有中
```

### 5. 平仓菜单增加平仓类型

平仓菜单新增下拉：

```text
止损
成本止损
移动止盈
手动平仓
```

已平仓模块新增字段：

```text
平仓类型
```

位置放在：

```text
状态 后面
```

### 6. 修复加仓/对冲不在主仓树显示的问题

原因是旧版新增加仓/对冲时，主仓编号默认还是 M1；如果 M1 已平仓，新订单会挂到已平仓主仓下面，所以主仓树不显示。

现在新增加仓/对冲时，主仓编号下拉只显示当前未平仓主仓，并默认选第一个活跃主仓。
