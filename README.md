# BTC交易管理系统

当前版本：`v0.6.5 表格/平仓/减仓细节修复版`

## v0.6.5 修复/新增

### 1. 主控界面字号恢复

总控页字号加大、行高加高，不再显得过小。

### 2. 平仓时间默认留空

平仓菜单字段：

```text
平仓价格
平仓时间
平仓订单号
```

其中 `平仓时间` 默认空白。

### 3. 减仓菜单增加平仓时间

减仓菜单字段中增加：

```text
平仓时间
```

默认空白，并放在平仓订单号前面。

### 4. 开仓时间默认留空

主仓 / 加仓 / 对冲开仓界面：

```text
开仓时间
```

默认空白，并放在开仓订单号前面。

### 5. 主仓平仓进入已平仓模块

主仓执行平仓后：

```text
状态切换为 已平仓
写入平仓价格/平仓时间/平仓订单号
进入 已平仓 模块
从主仓树中移除
```

### 6. 表格选中不再蓝底

表格点击条目后不再用蓝色底色遮挡文字颜色，选中行保持白底黑字。

### 7. 加仓/对冲/减仓编号增加主仓前缀

显示格式改为：

```text
M1-A1
M1-H1
M1-R1
```

减仓菜单第一排增加：

```text
主仓编号
```

### 8. 表格列宽双击自适应更紧凑

双击列分隔符后按当前显示内容自动适应列宽，并限制最大宽度，避免比文字宽太多。

### 9. 每次完成后列出测试点

后续每次修改完成都会明确列出你需要重点测试的地方。
