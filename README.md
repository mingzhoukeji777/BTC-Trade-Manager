# BTC交易管理系统

当前版本：`v0.6.9 表格单击清选中/已平仓修改框顺序修复版`

## v0.6.9 修复/新增

### 1. 表格空白处取消选中改为单击

原来需要双击表格空白处才会取消选中。

现在改为：

```text
单击表格空白区域 = 取消选中
```

保留列分隔符双击自动适应列宽功能。

### 2. 已平仓模块双击条目打开正确修改框

已平仓模块双击条目现在打开“修改已平仓”专用表单，不再走普通节点修改表单。

### 3. 已平仓模块增加“类型”列

已平仓表格现在显示：

```text
状态
类型
平仓类型
```

其中类型会显示：

```text
主仓
加仓
对冲
减仓
```

### 4. 修改已平仓表单中平仓类型放在类型下面

修改已平仓表单顺序改为：

```text
类型
平仓类型
平仓时间
平仓订单号
备注
```

保存时会更新：

```text
平仓类型
平仓时间
平仓订单号
备注
```
