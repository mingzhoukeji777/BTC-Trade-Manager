# BTC交易管理系统

当前版本：`v0.6.3 字段/减仓/状态指示修复版`

## v0.6.3 修复/新增

### 1. 页面标签增加编号字母

主页标签增加对应编号标识：

```text
主仓M
加仓A
对冲H
减仓R
```

### 2. 主仓自动计算手续费和盈亏平衡价

新增/修改主仓时，填写数量和开仓均价后会自动计算：

```text
手续费
盈亏平衡价
```

手续费单位按账户区分：

```text
OKX 币本位：BTC
Binance U本位：USDT
```

### 3. 主仓树状态指示灯

主仓树节点前增加状态球：

```text
🔴 已开仓 / 持有中
🟡 成本止损
🟢 移动止盈
⚪ 其他状态
```

### 4. 字体统一为微软雅黑

窗口、表格、弹窗、树状图统一使用：

```text
Microsoft YaHei / 微软雅黑
```

### 5. 已平仓表格重排

已平仓模块字段改为：

```text
编号
方向
状态
持仓量
盈亏
盈亏金额
开仓均价
平仓价格
手续费
开仓时间
平仓时间
开仓订单号
平仓订单号
备注
```

盈亏颜色：

```text
盈利：绿色
亏损：红色
```

### 6. 主仓增加资金费/手续费累计字段

主仓表格在平仓价格前增加：

```text
资金费率
手续费
累计资金费
累计手续费
```

并增加：

```text
开仓时间
平仓时间
```

### 7. 加仓 / 对冲增加资金费率和手续费

加仓、对冲表格在平仓价格前增加：

```text
资金费率
手续费
```

并增加：

```text
开仓时间
平仓时间
```

### 8. 减仓按钮移除，改成右键动作

顶部按钮已去掉：

```text
减仓
```

减仓只作为当前持有仓位的右键动作：

```text
选中持有中的加仓/对冲
右键
减仓
```

减仓模块表格字段改为：

```text
编号
方向
状态
减仓量
盈亏
盈亏金额
开仓均价
减仓价格
手续费
开仓时间
平仓时间
开仓订单号
平仓订单号
备注
```

### 9. 主仓 / 加仓 / 对冲增加开仓时间和平仓时间

字段放在订单号前：

```text
开仓时间
平仓时间
开仓订单号
平仓订单号
```
