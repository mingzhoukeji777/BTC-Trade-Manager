# BTC交易管理系统

当前版本：`v0.3.0 主仓树整页框架版`

## 核心定位

这是围绕完整牛熊周期设计的 BTC 长线主仓位管理系统。

它不是短线/波段/长线账户分类工具，而是分别管理：

```text
Binance U本位永续独立主仓
OKX 币本位永续独立主仓
```

两个账户独立管理，通过顶部下拉框切换。

## v0.3 已实现

### 1. 整页式界面

已去掉 v0.2 的左右分割主界面。

现在每个页面都是完整页面：

```text
总控
主仓管理
主仓树
节点明细
订单映射/待归类
风险总控
情景模拟
```

### 2. 主仓管理

支持右键：

```text
新增主仓
修改主仓
删除主仓
```

主仓字段：

```text
编号
方向
周期名称
状态
主仓数量
主仓均价
保证金
备注
```

方向、状态、编号均使用下拉菜单。

### 3. 主仓树

主仓树页面用于查看：

```text
M 主仓
├ A 加仓
├ H 对冲
├ TP 止盈
├ SL 止损
└ R 减仓
```

支持右键：

```text
新增主仓
新增节点
修改选中
删除选中
```

### 4. 节点明细

节点明细页面显示所有 A/H/TP/SL/R 节点。

支持右键：

```text
新增节点
修改节点
删除节点
```

节点字段：

```text
主仓编号
节点编号
类型
方向
状态
数量
开仓价
平仓价
手续费
资金费
盈亏平衡
成本止损
移动止盈
开仓订单号
平仓订单号
备注
```

节点编号、类型、方向、状态均使用下拉菜单。

### 5. 订单映射 / 待归类

支持右键：

```text
新增订单
修改订单
删除订单
```

订单字段：

```text
订单号
成交号
方向
动作
价格
数量
手续费
资金费
时间
匹配状态
备注
```

方向、动作、匹配状态均使用下拉菜单。

### 6. 风险总控

读取采集器：

```text
BTC实时通信系统\data\latest_snapshot.json
```

展示：

```text
采集状态
实时价格
交易所真实持仓
强平距离
节点数量
待归类订单数量
```

### 7. 情景模拟

当前支持输入目标 BTC 价格并记录模拟结果。

后续会扩展为：

```text
主仓盈亏
加仓分支盈亏
对冲分支盈亏
强平距离变化
移动止盈/减仓/解除对冲建议
```

### 8. UI 自动保存

手动调整窗口大小和位置后会自动保存，下次打开恢复。

保存文件：

```text
ui_settings.json
```

## 数据库核心表

```text
main_positions     主仓表

order_mappings     订单映射/待归类表

scenario_runs      情景模拟记录表
```
