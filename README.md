# BTC交易管理系统

当前版本：`v0.6.4 树状态灯/减仓逻辑/表格列宽修复版`

## v0.6.4 修复/新增

### 1. 主仓树状态灯改为图片圆点

主仓树不再用文字/emoji圆点，改成 ImageList 绘制的高亮圆形图标：

```text
红色：已开仓 / 持有中
黄色：成本止损
绿色：移动止盈
灰色：其他状态
```

### 2. 表格列宽支持自适应和保存

所有 DataGridView 表格：

```text
双击列分隔符：按内容自动适应列宽
手动调整列宽：关闭后保存
下次打开：恢复上次列宽
```

### 3. 已平仓盈亏金额自动计算

平仓时会写入：

```text
平仓价格
平仓时间
平仓订单号
盈亏金额
```

盈亏金额计算会扣除手续费。

### 4. 主仓右键增加减仓

主仓右键菜单增加：

```text
减仓
```

主仓平仓后也会记录平仓价格/时间/订单号。

### 5. OKX 币本位手续费计算修正

OKX 币本位手续费按 BTC 单位估算：

```text
手续费(BTC) = 数量(BTC) × 手续费率
```

Binance U本位手续费按 USDT 单位估算：

```text
手续费(USDT) = 数量 × 价格 × 手续费率
```

### 6. 减仓模块增加“减仓订单编号”

减仓表格字段增加：

```text
减仓订单编号
```

用于说明“减的是哪一单”。

### 7. 减仓弹窗优化

减仓弹窗：

```text
增加：减仓订单编号
自动填写：被减仓订单编号
去掉：方向选项
状态固定：已减仓
自动填写：开仓均价
自动填写：开仓订单号
自动填写：开仓时间
```

### 8. 平仓菜单增加平仓时间

平仓弹窗现在包括：

```text
平仓价格
平仓时间
平仓订单号
```

### 9. 主仓 / 加仓 / 对冲开仓界面增加开仓时间

新增/修改主仓、加仓、对冲时，开仓界面增加：

```text
开仓时间
```

加仓/对冲开仓界面去掉了：

```text
平仓订单号
```

平仓订单号只在平仓时填写。

## 仍需继续完善

减仓对原持仓的影响目前已能生成减仓记录和盈亏，但还没有完整反向扣减原持仓剩余数量/剩余成本。下一版应继续做：

```text
减仓后自动减少原加仓/对冲/主仓持仓量
按减仓部分计算已实现盈亏
保留剩余持仓成本
更新主仓风险汇总
```
