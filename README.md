# BTC交易管理系统

当前版本：`v0.7.1 主控放大/列宽居中/已平仓排序修复版`

## v0.7.1 修复/新增

### 1. 不再自动启动程序打断查看

后续修改完成后，默认只编译和发布新版 EXE，不自动打开程序。

### 2. 主控页面内容字体继续放大

主控页表格内容字体放大到 16 号加粗，行高加高到 56，方便查看。

### 3. 双击列分隔符后保持居中

修复双击列分隔符按标题宽度缩放后，单元格居中样式可能失效的问题。

现在双击列宽后会重新应用：

```text
表头居中
内容居中
当前列所有单元格居中
```

### 4. 固定主控页列宽

主控页新增固定布局逻辑：

```text
项目列约 22%
数值列约 30%
说明列占剩余空间
```

切换标签返回主控后，会按固定比例恢复，不再乱变。

### 5. 已平仓按平仓时间倒序排列

已平仓模块排序改为：

```text
最后平仓的排在最前面
平仓时间为空的排后面
同时间按内部 id 倒序
```
