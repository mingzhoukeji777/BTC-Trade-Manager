# BTC交易管理系统

当前版本：`v0.7.0 表格居中/列宽紧凑/已平仓全字段修改版`

## v0.7.0 修复/新增

### 1. 所有表格字体居中

统一改为：

```text
表格内容居中
表头文字居中
```

风险总控表格也补上了统一格式化。

### 2. 双击列宽按标题栏文字宽度收紧

原来双击列宽会按内容自动适应，还是偏宽。

现在改为：

```text
双击列分隔符 = 按标题栏文字宽度收紧
```

例如标题只有两个字，就不会因为内容很长把列拉得很宽。

### 3. 主控界面字体加大

主控页字体加大为 13 号加粗，行高加高。

说明栏也开启自动换行，并给说明栏更多显示空间。

### 4. 主控页停留时不再被定时刷新重置

以前停在总控页时，定时刷新会重新绑定总控表格，导致你刚调整列宽又被刷新覆盖。

现在改为：

```text
停留在总控页时，不自动重建总控表格
切到其他页时，仍正常刷新风险/数据
手动点“刷新”仍可刷新全部
```

### 5. 已平仓双击修改框支持修改全部条目字段

已平仓模块双击条目后，现在可以修改：

```text
编号
方向
状态
类型
平仓类型
持仓量
盈亏
盈亏金额
开仓均价
平仓价格
手续费
开仓时间
平仓时间
开仓订单号
平仓订单号
备注
```

保存后会写回对应的：

```text
主仓记录
加仓/对冲/减仓记录
```
