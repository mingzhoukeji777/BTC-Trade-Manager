# BTC交易管理系统

当前版本：`v0.7.3 主控字号回调/列宽保持/表头居中/标签不加粗版`

## v0.7.3 修复/调整

### 1. 主控页面字体回调

上一版主控内容字体过大，本版调整为：

```text
主控内容字体：14号
主控内容字重：不加粗
主控表头字体：13号加粗
主控行高：42
```

### 2. 修改开仓时间/平仓时间后列宽不再变化

修复编辑保存后刷新表格导致列宽变化的问题。

现在编辑保存时会：

```text
保存前捕获所有表格当前列宽
刷新数据
刷新后恢复原列宽
再保存列宽设置
```

覆盖路径包括：

```text
已平仓修改
主仓修改
加仓/对冲/减仓修改
平仓
成本止损/移动止盈
订单映射修改
删除记录
情景模拟
设置保存
```

### 3. 自适应列宽后表头仍保持居中

双击列分隔符后，现在会重新强制：

```text
关闭表头视觉样式覆盖
表头居中
内容居中
所有列 HeaderCell 居中
当前列所有单元格居中
保存列宽
```

### 4. 标签文字不加粗

标签页文字从加粗改为普通字重：

```text
Microsoft YaHei 10 Regular
```

### 5. 本地 EXE

本次没有启动程序。

新版文件：

```text
C:\Users\xiaozhu\Desktop\BTC交易管理系统\BTC交易管理系统_v0.7.3.exe
```

如果旧版仍在运行，先关闭旧版，再运行这个新版。
